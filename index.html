"Nunca é apenas código"

Estou ansioso para compartilhar minha visão e colaborar com você para alcançar seus objetivos práticos e criativos.

Neste portfólio você será apresentado minhas experiências e habilidades.

Tecnologias:

Java JavaScript SQL/MongoDB CSS 3 HTML 5 Figma Git/GitHub

Java

Iniciei a minha jornada com esta linguagem de programação. Meu conhecimento em Java vai desde funções simples até frameworks e bibliotecas complexas. Meu portfólio de programador Java conta com aplicações Desktop (JavaSwing e JavaFX) até soluções Web completas (SpringBoot/SpringMVC). Tenho domínio sólido de Programação Orientada a Objetos, Integração com Banco de Dados (JPA Hibernate e JDBC), criação de API e ferramentas de depuração e teste.

JavaScript

Atualmente é a linguagem de programação que mais uso e estudo, tenho imenso carinho por esta tecnologia e suas possibilidades. Meu conhecimento em JavaScript inclui soluções simples de interatividade, mas também abrange NODE js, JQuery e BootsTrap. Também tenho sólido conhecimento com consumo de API, manipulação de dados/eventos, DOM e ferramentas de depuração e teste como DevTools.

DataBases

Meu conhecimento em bancos de dados é voltado a bancos relacionais como MySql, embora também tenha estudado e desenvolvido soluções usando MongoDB e Google Firebase. Tenho ampla experiência com modelagem e manutenção de bancos relacionais, que vai desde UML, análise de dados, segurança de dados, plano de backup e testes de desempenho. Meu portfólio conta com algumas aplicações Desktop e Web integradas com banco de dados via frameworks como JPA Hibernate e Spring JPA.

CSS 3

Tenho domínio satisfatório de CSS, sou um amante de um bom design e considero o estilo uma dos pilares de uma boa experiência. Dentre meus conhecimentos em CSS posso destacar as animações, responsividade e integração com JavaScript. Vale lembrar que realizei trabalhos acadêmicos voltados para acessibilidade e usabilidade, que fazem grande proveito desta tecnologia.

HTML 5

Minhas skills com HTML vão desde SEO e marcação semântica, formulários e acessibilidade, até compatibilidade com navegadores e requisições WEB. Esta tecnologia acompanha quase todos os projetos de meu portfólio e considero um conhecimento essencial.

Design

Dentre meus conhecimentos com design, se destacam algumas ferramentas de projeto como Figma e Photoshop. Minha experiência passa por UX/UI, prototipagem e desenvolvimento de wireframes. Também possuo habilidades avançadas para edição de imagem, video e áudio com ferramentas como Canva, Reaper e FL Studio.

Git / GitHub

Estou bem familiarizado com versionamento de código, ao longo de minha experiência tive contato com diversas opções existentes no mercado. A ferramenta que mais uso atualmente é o GitHub e suas ferramentas como GitHub Pages e GitHub Desktop, meu repositório de código pode ser encontrado na sessão anterior da página.
