FilmeDex

Java (SpringMVC), MySql, HTML, CSS, JavaScript (BootsTrap)

Este projeto é um catálogo de filmes interativo que lista os registros cadastrados no banco de dados, e também possibilita que o usuário edite, remova ou escreva uma análise com nota para um determinado filme. Foi utilizado Bootstrap para aprimorar a responsividade e SpringMVC para gerenciar as rotas e páginas. Foi criada também uma classe REST para gerenciar a API do projeto, que possibilita fazer todas as operações citadas anteriormente através de requisições HTTP. Filmes e Análises são relacionados através de chaves estrangeiras no banco de dados.
